# MovieRecommender-Analysis
A Python-based movie recommendation and analytics project using the MovieLens dataset.
This project analyzes movie ratings, identifies popular movies, and recommends movies based on genres.

# Features
Genre popularity analysis
Decade-wise rating trends
Rating distribution analysis
Advanced popularity scoring system
Genre-based movie recommendation system
Interactive Streamlit interface

# Source:
MovieLens Dataset

# Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Streamlit
